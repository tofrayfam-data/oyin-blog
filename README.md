# My Child And I With Oyin 👩‍👧✨

A purpose-driven parenting platform focused on raising emotionally healthy, spiritually grounded, and confident children.

This platform provides guidance, real-life parenting insights, and faith-based encouragement for parents navigating modern parenting challenges.

---

## 🌍 Project Overview

"My Child And I With Oyin" is a web-based platform designed to:

- Share practical parenting advice
- Provide emotional and spiritual guidance for families
- Deliver engaging blog content for parents and teens
- Promote healthy parent-child relationships

---

## 🎯 Key Features

- 📚 Dynamic blog system (JavaScript-powered)
- 🔍 Search and filter functionality by category
- ⭐ Featured and trending posts section
- 📄 Individual post pages with full content rendering
- 🎥 Embedded media (PDF resources, guides)
- 💡 Clean and responsive UI design

---

## 🛠 Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- JSON-like data structure (posts.js)
- Google Drive PDF embedding

---

## 🧠 Key Highlights

- Dynamic content rendering without backend dependency
- Optimized blog filtering and pagination system
- Clean UI/UX focused on readability and engagement
- Modular JavaScript structure for scalability
